# 📊 Student Attendance Tracker (GUI)

## 📌 Description
A Python-based GUI application that helps students track their attendance and predict how many classes they need to attend or can skip. 
This is just a prototype, we can make it look more attractive by adding more colours and UIs.

---

## 🚀 Features
- Calculate attendance percentage
- Predict required classes to reach 75%
- Suggest how many classes can be bunked
- Simple and user-friendly interface

---

## 🛠 Tech Stack
- Python
- Tkinter (GUI Library)

---

## ▶️ How to Run

1. Install Python (3.x)
2. Clone the repository:
   git clone https://github.com/your-username/attendance-tracker.git

3. Navigate to the folder:
   cd attendance-tracker

4. Run the program:
   python main.py
